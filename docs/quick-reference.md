## The Core Loop We're Building

```
Producer allocates ‚Üí Employee confirms ‚Üí Manager approves ‚Üí Dashboard updates
     (web)              (Slack DM)         (Slack/web)         (web)
```

---

## Key Pain Points We're Solving

1. External tool = forgotten ‚Üí **Slack DMs**
2. No real-time visibility ‚Üí **Live dashboard**
3. Rubber-stamping ‚Üí **Flag when actual = planned**
4. Too late to course-correct ‚Üí **Weekly approvals**
5. Hidden UI buttons ‚Üí **One page, no hunting**

---

## Design Principles

1. Trust = Accuracy + Adoption + Auditability
2. One page, no hunting
3. Company-wide view first
4. Proactive, not reactive
5. Meet people where they are (Slack!)
6. Quick context, not drilling (project snippets)
7. LLM-assisted, not LLM-dependent
